This is a set of files to accompany a tutorial for implementing subtitles with PoodLL/VideoEasy on Moodle.

Download and unzip (or clone) this repository to get :
i) a sample video file (zombie_problems.mp4)
ii) a sample subtitle file (zombie_problems.vtt)
iii) a subtitle video player template for PoodLL/VideoEasy
iv) a remove link template for PoodLL/VideoEasy

Any questions? Post them on the PoodLL Forum at Moodle.org
